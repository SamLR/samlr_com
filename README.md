# My website

Because otherwise I forget how this all works...

## Running

Running the server
```bash
$ pipenv run server
```

Building the CSS
```bash
$ gulp styles       # one time
$ gulp              # By default will watch the sass
```

Static build
```bash
$ guld build  # Makes everything
```

## Publishing

```bash
$ ./publish.sh
```

## Installing

Currently using `pipenv` and `npm`

```bash
$ pipenv install
$ npm install -D
```
